# Documentacao

Esta pasta concentra arquivos adicionais de apoio ao projeto, com foco em arquitetura, pipeline, execução e organização de artefatos gerados

## Arquivos disponíveis

- `pipeline-diagram.mmd`: diagrama do fluxo principal de dados.
- `guia-execucao.md`: Passo a passo para executar o pipeline.

## Visualização dos diagramas

O arquivo .mmd pode ser visualizado no seguinte site:

- https://mermaid.live

## Objetivo desta pasta

O objetivo da documentação é complementar o README principal com materiais mais operacionais e técnicos, facilitando o seguinte:

- entendimento rapido da arquitetura;
- reprodução do pipeline;
- consulta dos artefatos gerados;
- apresentação acadêmica do projeto na cadeira Big Data.
